Introduction

L'API CS2 Players fournit des informations détaillées sur les joueurs professionnels de Counter-Strike 2. Elle permet d'accéder aux données des joueurs, leurs équipes, et plus encore.

URL de base : https://cs2-api.onrender.com/api

Format de réponse : JSON

Endpoints

Liste des joueurs

GET
/players

Joueur spécifique

GET
/players/{idPlayer}

Récupère les informations d'un joueur spécifique.

Réponse

{
  "id": "donk",
  "nickname": "donk",
  "team": "Team Spirit",
  "nationality": "/images/flags/ru.svg",
  "age": 18,
  "trophies": 8,
  "kd": 1.35
}

Liste des équipes

GET
/teams

Récupère la liste de toutes les équipes.

Réponse

[
  "Team Spirit",
  "FaZe Clan",
  "Natus Vincere",
  "Team Vitality",
  "G2 Esports",
  "MOUZ",
  "Virtus.pro",
  "Team Falcons"
]

Joueurs d'une équipe

GET
/teams/{team}/players

Récupère tous les joueurs d'une équipe spécifique.

Réponse

[
  {
    "id": "donk",
    "nickname": "donk",
    "team": "Spirit",
    ...
  },
  ...
]

Recherche

GET
/search?query={team or nickname}

Recherche des joueurs par nom ou équipe.

Réponse

[
  {
    "id": "donk",
    "nickname": "donk",
    "team": "Spirit",
    ...
  },
  ...
]

Exemples d'utilisation

Exemple d'url

# Liste des joueurs
https://cs2-api.onrender.com/api/players

# Joueur spécifique
https://cs2-api.onrender.com/api/players/donk

# Joueurs d'une équipe
https://cs2-api.onrender.com/api/teams/Spirit/players

# Recherche
https://cs2-api.onrender.com/api/search?query=donk
